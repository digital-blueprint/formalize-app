# Agents Instructions

## Important Notes for AI Assistants

1. Forms are present in `src/modules/`
2. Don't update files in `dist/` as they are generated from the source files in `src/`
3. If you need to update the `CHANGELOG.md`, make sure to follow the existing format and include the version number and a clear description of the changes made
4. In forms always put field descriptions between the field name and input field, so that the order of information is fitting for screen readers.
